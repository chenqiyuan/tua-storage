快速上手 →

多端统一 api

支持 web 端、小程序端、React-Native、Node.js 端。

支持数据同步

没有数据怎么办，自动同步之。

数据过期逻辑

支持设定数据过期时间，再次读取时调用同步函数自动同步。

自动清理过期数据

启动时以及定时清理过期数据。

支持永久保存

数据可以设定为永不过期。

支持批量操作

接口同时支持传入对象或数组。
